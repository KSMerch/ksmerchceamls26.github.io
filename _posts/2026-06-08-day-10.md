what_i_learned: |
  Today was our first full lab-focused day, and we spent most of the time working directly with machine learning models. We did a deeper dive into 1D Convolutional 
  Neural Networks (1D CNNs) and how they are used to process sequential data like ECG signals. I learned how these models are able to pick up patterns over time, 
  which is important for understanding heart activity in our project. We also spent time trying to program a baseline 1D CNN model. This helped me understand how the 
  different parts of a neural network fit together. The model starts by detecting simple patterns in the ECG signal, such as small changes in the waveforms. It then 
  goes through layers that refine and clean the information while reducing its size so it can be processed more efficiently. After that, it learns more complex 
  patterns based on what it has already identified. At the end, the data is flattened into a single list of features, and the model uses those features to make 
  predictions across five different heart condition categories. Each output represents the likelihood that a specific condition is present in the ECG. Overall, this 
  helped me see how a basic neural network is structured and how raw ECG data is gradually transformed into meaningful predictions.
