what_i_learned: |
  Today was a full lab day focused on programming and refining the training code for our 1D CNN model. Most of our time was spent working on the code that actually trains the model using the ECG data. This helped me 
  better understand what happens after the data has been prepared and loaded into the model. We configured the model to train using three different random seeds and run for 60 epochs each. I learned that using 
  multiple seeds helps test whether the model's results are consistent or if they vary depending on the random starting conditions. This is especially important for our project because we want to make sure that any 
  conclusions we draw about model performance and fairness are reliable. I also learned that an epoch represents one complete pass through the training dataset, so running for 60 epochs gives the model many 
  opportunities to learn patterns from the ECG recordings. Throughout the day, I gained a better understanding of how the training process is set up and why reproducibility is such an important part of machine 
  learning research. Rather than just building a model, we are also making sure that the results can be trusted and evaluated properly.

blockers: |
  I encountered a shape mismatch error between the flattened CNN output and the fully connected layer. The CNN was producing 15,680 features, but the linear layer was configured to accept 16,000 features. It took me 
  a minute to figure this out though, but after correcting the input size of the linear layer, the model ran successfully.
